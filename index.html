One-time setup — takes about 3 minutes
1
Go to console.firebase.google.com → click Add project → name it anything (e.g. editorial-hub) → disable Google Analytics → Create project.
2
Inside the project → click Build → Realtime Database → Create database → choose any location → start in Test mode → Enable.
3
Go to Project Settings (gear icon) → scroll to Your apps → click the </> Web icon → register app → copy the firebaseConfig object values into the fields below.

Your config is saved in your browser. Free Firebase tier supports up to 100 simultaneous users.
